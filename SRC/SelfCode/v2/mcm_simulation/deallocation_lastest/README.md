#重大发现
关于在实验室航行器加载不出来（只是m10任务以及和像我后来做的由m10衍生的任务出现这种情况）。   
**原因**：
>当前意识到的是无线网络没有连接。    
**分析**：    
>不联网的话，本地IP获取不到，localhost得不到值。这个类似于在本地用SSH搭建服务器一样。
**对策**:
>如果上面的分析是对的话，那么我插根网线和其它电脑链接上应该就能解决。待实验。
**结论**:
>已验证，上述分析正确。
